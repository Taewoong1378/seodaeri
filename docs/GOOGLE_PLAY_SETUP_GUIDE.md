# Google Play Console 설정 가이드 (서대리님용)

---

## 전체 흐름 요약

```
서대리님이 하실 일 (약 20분)
├── 1단계: Google Play Console 개발자 계정 등록 ($25)
├── 2단계: 개발자 초대하기
└── 3단계: 앱 만들기

개발자가 할 일 (이후)
├── 앱 빌드 및 업로드
├── 스토어 등록정보 입력
└── 심사 제출
```

---

## 1단계: Google Play Console 개발자 계정 등록

### 준비물

- Google 계정 (Gmail)
- 신용카드 또는 체크카드 ($25 = 약 3만 3천원, **일회성**)
- 휴대폰 (본인 인증용)

### 진행 방법

1. **Google Play Console 접속**

   - https://play.google.com/console 에 접속
   - Google 계정으로 로그인

2. **개발자 계정 만들기**

   - `개발자 계정 만들기` 버튼 클릭
   - 계정 유형 선택:
     - **개인**: 개인 명의로 등록
     - **조직**: 사업자등록증 필요
   - 👉 처음에는 **개인**으로 시작해도 됩니다

3. **개발자 정보 입력**

   - 개발자 이름 (스토어에 표시됨)
   - 이메일 주소
   - 전화번호
   - 웹사이트 (선택사항)

4. **본인 인증**

   - 휴대폰 번호로 인증 코드 받기
   - 신분증 인증이 필요할 수 있음 (개인 계정의 경우)

5. **결제**

   - 등록비 $25 USD (일회성, 매년 내지 않아도 됨!)
   - 결제 완료 후 **즉시** 또는 **48시간 내** 승인

6. **승인 확인**
   - 이메일로 승인 완료 알림이 옵니다
   - https://play.google.com/console 에서 확인 가능

---

## 2단계: 개발자 초대하기

계정 등록이 완료되면 개발자를 초대해주세요.

### 진행 방법

1. https://play.google.com/console 접속
2. 좌측 메뉴에서 `사용자 및 권한` 클릭
3. `새 사용자 초대` 버튼 클릭
4. 개발자 정보 입력:
   - **이메일 주소**: 개발자 Google 계정 이메일
5. **권한 설정**:
   - `관리자` 체크 ⬅️ 중요!
   - 또는 세부 권한을 선택할 수 있음
6. `사용자 초대` 클릭

> 💡 **개발자에게 전달할 정보**
>
> - "Google Play Console에 초대 보냈어요"
> - "이메일 확인하시고 초대 수락해주세요"

---

## 3단계: 앱 만들기

앱 "껍데기"를 먼저 만들어두면 개발자가 빌드를 업로드할 수 있습니다.

### 진행 방법

1. https://play.google.com/console 접속
2. `모든 앱` 클릭
3. `앱 만들기` 버튼 클릭
4. 정보 입력:

| 항목           | 입력값         |
| -------------- | -------------- |
| 앱 이름        | `gulim`        |
| 기본 언어      | 한국어 – ko-KR |
| 앱 또는 게임   | 앱             |
| 유료 또는 무료 | 무료           |

5. **선언** 체크:
   - [ ] 개발자 프로그램 정책 동의
   - [ ] 미국 수출법 준수 동의
6. `앱 만들기` 클릭

## 서대리님이 하실 일 체크리스트

### 지금 당장

- [ ] Google Play Console 개발자 계정 등록 ($25 결제)
- [ ] 계정 승인 대기 (보통 즉시~48시간)

### 승인 후

- [ ] 개발자 초대 (관리자 권한)

### 심사 제출 전 (개발자에게 전달)

- [ ] 앱 설명 텍스트 (4000자 이내)

> 💡 앱 아이콘, 스크린샷, 키워드 등 나머지는 개발자가 처리합니다.

## 자주 묻는 질문

### Q: 등록하는데 얼마나 걸려요?

결제 후 보통 즉시 승인됩니다. 신분증 인증이 필요하면 48시간까지 걸릴 수 있어요.

### Q: $25는 매년 내야 하나요?

**아니요!** Google Play는 **일회성** $25만 내면 됩니다. (Apple은 매년 $99)

### Q: 개인으로 등록하면 문제가 있나요?

없습니다. 나중에 조직 계정으로 전환할 수 있어요.

### Q: 개발자한테 관리자 권한 줘도 괜찮나요?

네, 괜찮습니다. 관리자여도 **계정 소유권**은 등록한 분에게 있어요.
결제 정보 변경, 계정 삭제 등은 계정 소유자만 할 수 있습니다.

## Apple vs Google 비교

| 항목        | Apple App Store | Google Play Store |
| ----------- | --------------- | ----------------- |
| 등록비      | $99/년 (매년)   | $25 (일회성)      |
| 승인 시간   | 24~48시간       | 즉시~48시간       |
| 심사 기간   | 1~3일           | 몇 시간~3일       |
| 개발자 초대 | 2곳에서 각각    | 1곳에서만         |
