# 굴림 - 눈덩이처럼 불려가는 내 자산

## 서비스 소개

**굴림**은 직장인 투자자를 위한 투자 기록 서비스입니다.

눈덩이를 굴리듯, 작은 시작이 큰 자산으로 불어나는 과정을 기록하고 시각화합니다.

복잡한 설정 없이 바로 시작하거나, 유튜브 채널 [서대리TV](https://www.youtube.com/@서대리TV)의 구글 스프레드시트 템플릿과 연동해서 사용할 수 있습니다.

> **"작게 시작해서, 꾸준히 굴려가세요"**

## 이런 분들을 위해 만들었습니다

- 🚀 **간편하게 시작하고 싶은 분** - 복잡한 설정 없이 바로 투자 기록을 시작하고 싶은 분
- 📊 **서대리 시트 사용자** - 기존 서대리 투자 템플릿을 모바일에서도 보고 싶은 분
- 💼 **직장인 투자자** - 바쁜 일상 속에서도 투자 기록을 놓치고 싶지 않은 분
- 💰 **배당 투자자** - 배당금이 눈덩이처럼 불어나는 걸 확인하고 싶은 분
- ❄️ **복리의 마법을 믿는 분** - 시간이 만들어주는 자산 성장을 기록하고 싶은 분

## 핵심 기능

### 1. 바로 시작하기

스프레드시트 없이도 바로 시작할 수 있습니다. Google 계정으로 로그인하면 즉시 투자 기록을 시작합니다.

- 복잡한 설정 없이 간편 시작
- 기존 서대리 시트 연동도 지원
- 새 시트 원클릭 생성

### 2. 대시보드

내 자산이 얼마나 굴러왔는지 한눈에 확인합니다.

- **계좌현황(누적)** - 총 자산, 투자원금, 수익률, 포트폴리오 비중
- **계좌현황(올해)** - 월별 손익, 주요 지수 대비 수익률 비교
- **배당현황** - 월별/연도별 배당금 추이, 누적 배당금

### 3. 포트폴리오 관리

보유 종목을 직접 관리하고 실시간 시세를 확인합니다.

- 종목 추가/수정/삭제
- 실시간 주가 조회
- 포트폴리오 비중 시각화

### 4. OCR 자동 인식

이미지만 찍으면 자동으로 데이터를 입력합니다.

- 배당금 입금 내역 자동 인식
- 거래 내역 이미지 분석
- 여러 건의 배당도 한 번에 입력

### 5. 배당 분석 차트

배당 눈덩이가 얼마나 커졌는지 다양한 관점에서 분석합니다.

- 연도별 배당금 비교
- 누적 배당금 추이
- 12개월 롤링 배당금
- S&P500 대비 수익률 비교

### 6. 공유 기능

굴려온 성과를 SNS에 공유할 수 있습니다.

- 차트 이미지 저장
- 인스타그램/트위터 공유용 이미지 생성

## 서비스 특징

### 간편한 시작

스프레드시트 연동 없이도 바로 사용할 수 있습니다. 원하시면 언제든 서대리 시트와 연동할 수 있습니다.

### 내 데이터는 내 것

모든 투자 데이터는 안전하게 저장됩니다. 서대리 시트 연동 시 내 Google 계정의 스프레드시트에도 저장됩니다.

### 무료로 사용

실시간 주가와 환율을 무료로 제공합니다.

---

## 사용 방법

### Step 1. 로그인

Google 계정으로 로그인합니다.

### Step 2. 시작 방법 선택

- 바로 시작하기 → 즉시 투자 기록 시작
- 기존 서대리 시트가 있다면 → Google Drive에서 선택
- 처음이라면 → '새 시트 만들기' 클릭

### Step 3. 굴리기 시작

대시보드에서 내 투자 현황을 확인하고, 포트폴리오와 배당 기록을 추가합니다.
작은 눈덩이부터 굴려보세요.

## 왜 "굴림"인가요?

> "복리는 세계 8번째 불가사의다. 이해하는 자는 벌고, 이해하지 못하는 자는 지불한다."
> — 알버트 아인슈타인 (출처 불명)

**굴림**은 눈덩이 효과(Snowball Effect)에서 영감을 받았습니다.

처음엔 작은 눈덩이지만, 꾸준히 굴리면 어느새 거대한 눈사람이 됩니다.
투자도 마찬가지입니다. 작은 금액이라도 꾸준히 기록하고, 배당을 재투자하면 시간이 만들어주는 복리의 마법을 경험할 수 있습니다.

**굴림**과 함께 당신의 자산 눈덩이를 굴려보세요.
