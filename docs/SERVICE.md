# 굴림 - 눈덩이처럼 불려가는 내 자산

## 서비스 소개

**굴림**은 직장인 투자자를 위한 투자 기록 서비스입니다.

눈덩이를 굴리듯, 작은 시작이 큰 자산으로 불어나는 과정을 기록하고 시각화합니다.

유튜브 채널 [서대리TV](https://www.youtube.com/@서대리TV)에서 제공하던 구글 스프레드시트 투자 기록 템플릿을 모바일에서도 편리하게 사용할 수 있도록 만든 앱입니다.

> **"작게 시작해서, 꾸준히 굴려가세요"**

---

## 이런 분들을 위해 만들었습니다

- 📊 **서대리 시트 사용자** - 이미 서대리 투자 템플릿으로 기록 중인 분
- 💼 **직장인 투자자** - 바쁜 일상 속에서도 투자 기록을 놓치고 싶지 않은 분
- 📱 **모바일 우선 사용자** - PC보다 스마트폰이 편한 분
- 💰 **배당 투자자** - 배당금이 눈덩이처럼 불어나는 걸 보고 싶은 분
- ❄️ **복리의 마법을 믿는 분** - 시간이 만들어주는 자산 성장을 기록하고 싶은 분

## 핵심 기능

### 1. 내 시트 연동

기존에 사용하던 서대리 투자 템플릿을 그대로 연결합니다. 새로운 사용자라면 템플릿을 자동으로 복사해서 시작할 수 있습니다.

- Google 계정으로 간편 로그인
- 기존 시트 자동 연동
- 새 시트 원클릭 생성

### 2. 대시보드

내 자산이 얼마나 굴러왔는지 한눈에 확인합니다.

- **총 자산** - 현재 보유 자산 총액
- **총 수익률** - 투자 성과 요약
- **배당금 현황** - 월별/연도별 배당금 추이
- **포트폴리오** - 종목별 비중 시각화

### 3. 배당 분석 차트

배당 눈덩이가 얼마나 커졌는지 다양한 관점에서 분석합니다.

- 연도별 배당금 비교
- 누적 배당금 추이
- 배당 수익률 vs S&P500 비교
- 12개월 롤링 배당금

### 4. 공유 기능

굴려온 성과를 SNS에 공유할 수 있습니다.

- 차트 이미지 저장
- 인스타그램/트위터 공유용 이미지 생성

## 서비스 특징

### 내 데이터는 내 것

모든 투자 데이터는 **내 Google 계정의 스프레드시트**에 저장됩니다. 서비스가 종료되더라도 데이터는 영원히 내 것입니다.

### 무료로 사용

주식 시세 조회에 비싼 API를 사용하지 않습니다. Google 스프레드시트의 `GOOGLEFINANCE` 함수를 활용해 실시간 시세를 무료로 제공합니다.

### 서대리 시트와 완벽 호환

기존 서대리TV 투자 템플릿의 모든 수식과 구조를 그대로 활용합니다. 앱과 시트를 병행해서 사용해도 문제없습니다.

## 사용 방법

### Step 1. 로그인

Google 계정으로 로그인합니다.

### Step 2. 시트 연동

- 기존 서대리 시트가 있다면 → Google Drive에서 선택
- 처음이라면 → '새 시트 만들기' 클릭

### Step 3. 굴리기 시작

대시보드에서 내 투자 현황을 확인하고, 매매 기록을 추가합니다.
작은 눈덩이부터 굴려보세요.

---

## 향후 계획

- **AI 자동 입력** - 매매 체결 스크린샷만 찍으면 자동으로 데이터 추출
- **한국 주식 종목 검색** - KRX API 연동으로 종목명만 입력하면 종목코드 자동 완성
- **거래 내역 관리** - 앱에서 직접 매수/매도/배당 기록 추가
- **알림 기능** - 배당금 입금일, 목표 달성 알림
- **커뮤니티** - 다른 투자자들의 눈덩이 구경하기 (익명)

## 왜 "굴림"인가요?

> "복리는 세계 8번째 불가사의다. 이해하는 자는 벌고, 이해하지 못하는 자는 지불한다."
> — 알버트 아인슈타인 (출처 불명)

**굴림**은 눈덩이 효과(Snowball Effect)에서 영감을 받았습니다.

처음엔 작은 눈덩이지만, 꾸준히 굴리면 어느새 거대한 눈사람이 됩니다.
투자도 마찬가지입니다. 작은 금액이라도 꾸준히 기록하고, 배당을 재투자하면 시간이 만들어주는 복리의 마법을 경험할 수 있습니다.

**굴림**과 함께 당신의 자산 눈덩이를 굴려보세요.
